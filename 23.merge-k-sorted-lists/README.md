[21题](../21.merge-two-sorted-lists/21.合并两个有序链表.py)已经探讨了合并两个有序链表的方法，如果要合并k个有序链表的话，有一个很直白的方法，就是用一个变量
来维护合并后的链表，然后一个for循环，将待合并的每个链表与这个链表合并。

上面这个方法可以进行一个优化，不要顺序合并，而是分治合并，将k个链表两两配对合并，第一轮之后就只剩下k/2个链表，第二轮后剩下k/4个链表，这样一直重复下去，最后会得到最终的链表。

最后还有一个方法，我们需要维护一个list，这个list里面分别是这个k个链表中最前面的一个，即最小的那个元素，然后从这k个元素中选最小的那一个合并到最终链表中，选择之后再将这个链表的next item加入list中，直到next为None，然后重复这个过程直到list为空。这个list我们可以使用优先队列。

python中要使用`heapq`必须要`ListNode`重写`__lt__`方法，这里不太方便，遂用golang实现优先队列的方法(Golang也不是很方便呀...)
